.. _transfer_to_gcu:

####################################
CUDA 代码一键迁移
####################################

.. contents:: 目录

.. warning::
    此功能依赖 cuda 版本的PyTorch，需要用户手动安装。


为了方便用户代码从 CUDA 迁移至 GCU 平台, torch_gcu 支持一键迁移功能, 仅需要在用户代码调用之前调用

.. code-block:: python

    from torch_gcu import transfer_to_gcu

即可支持用户代码在 GCU 硬件上执行

========
使用示例
========

.. code-block:: python
    :linenos:
    :emphasize-lines: 3,4

    import torch
    try:
        import torch_gcu # import torch_gcu
        from torch_gcu import transfer_to_gcu # import transfer_to_gcu
    except Exception as e:
        print(e)

    # user code
    print(f"torch.cuda.is_available(): {torch.cuda.is_available()}")

    a_tensor = torch.ones(3, 3).cuda()
    b_tensor = torch.ones(size=(3, 1), device="cuda")

    add_out = torch.add(a_tensor, b_tensor)
    print(add_out)

.. warning::
    torch.jit.script 模块被整体禁用
